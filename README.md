# TrackFit Ultimate

App de entrenamiento personalizada para fútbol.

## Desarrollo local

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy en Vercel

1. Subí este repo a GitHub
2. Entrá a vercel.com → Add New Project
3. Importá el repo (Vercel detecta Vite automáticamente)
4. Deploy
